此程序用于替换Windows 8.1/10中微软五笔的码表，以使你可以使用自己习惯的码表。

说明

  1. 微软五笔最长只支持4码，且不支持z键。这是软件本身限制，本人无法改进。
  2. 码表格式可参见后文附图，编码在前，词条在后，词条与编码、编码与编码用空格分隔。如果你的码表不是这种格式，请先将其转为此格式，可以考虑使用多多码表编辑器。
  3. 为了安装成功，请确保微软五笔未处于使用状态，否则安装程序会因码表被锁定而失败。建议您先将微软五笔卸载，重启Windows，然后再运行本程序，安装完码表后再重新装上微软五笔。
  4. 下面的附图是v1.0的，但软件已更新至v1.1，主要改进了提示消息，及添加对Windows 10周年版的支持。

用法：

  1. 点“加载...”将码表加载到程序。
  2. 点“安装...”将码表安装到系统。
  3. 现在你切换到“微软五笔”，会现现码已经变成你加载的码表了。
  4. 你也可以点“保存...”将转换好的二进制格式的微软码表保存起来，供以后加载。

注意：本程序需要.NET Framework 4.5才能运行，请在Windows 8.1/10控制面板中将.NET Framework打开。

注意：安装码表到系统需要在Windows 8.1/10上才能成功，且需要管理员权限。

程序支持加载两格式的码表。

  • 可以是二进制格式，这是本程序生成的，或微软五笔自带的*.lex文件。请确保加载的二进制码表没有损坏，否则安装后可能发生不可预知的错误。
  • 可以是文本格式。使用类似于QQ五笔的格式，请以Unicode方式保存。参考下图的说明和Tables中的示例码表。

为方便使用，及起示例作用，程序已自带3套五个码表，它们在Tables下面。另外为起备份作用，附带了微软五笔自带的码表，取名ChsWubi.backup.lex/ChsWubiNew.backup.lex。

  1. 极点极爽词库4.3版_修订版（其中“修订”指改了符号的编码，见后面的说明）
  2. Windows经典五笔输入法小词库86版和98版
  3. 取自大一统2014的新世纪五笔词组版和单字版

各码表仅限个人使用，版权归原软件公司所有，不得用于商业目的。

关于符号编码的说明

由于不能以“z”开头，所以将引导标点的“zz”改为“co”。

下面几个字母串后面也有“z”，将它们全部换成“o”：

zzz -> coo
zzzs -> coos
zzzy -> cooy

带声调的a、o、e、i、u和ü原编码长度为5，不支持，因此做以下修改：

zzpya -> copa
zzpye -> cope
zzpyi -> copi
zzpyo -> copo
zzpyu -> copu
zzpyv -> copv

极点自带的“极爽词库4.3版”码表和86、98没有以“co”开头的其他词条，因此上述处理不会引起任何冲突。

新世纪版（包括单字版和词组版）在这两个词条上和符号设置重码了，汉字会显示在前面，标点跟随其后。

copy 䮤
copi 遤 騌

由于不受支持，因此删除了极点生僻字标记（如“囧”原来为“~囧”）和变量（如原来可以用“now”输出当前时间）。
